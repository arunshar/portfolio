Physics-informed AI for reliable spatial reasoning.

I build physics-informed AI systems that make spatial reasoning reliable for real-world deployment, from detecting GPS spoofing to enable safer autonomous navigation, to modeling energy-efficient vehicle routing, to downscaling climate data for coastal flood risk assessment.

My research fuses deep generative models, diffusion models, transformers, physics-informed learning, and geostatistical priors to address data incompleteness, distribution shift, and violations of physical laws in spatial machine learning.

I earned my Ph.D. in Computer Science from the University of Minnesota, Twin Cities, advised by Prof. Shashi Shekhar, with committee members Prof. Vipin Kumar, Prof. Ravi Janardan, and Prof. Ying Song. At Esri, I shipped a maritime anomaly detection pipeline on AWS that improved classification accuracy from 55% to 73% and reduced API latency by 30%.

My technical work spans Python, Java, R, SQL, Scala, C/C++, Julia, Rust, Go, JavaScript, TypeScript, PyTorch, TensorFlow, JAX, XGBoost, MLflow, Hugging Face, LangChain, vector databases, PySpark, Kafka, Docker, Kubernetes, SageMaker, Lambda, ECS, Bedrock, vLLM, and SGLang.

Selected Projects

Computer Vision / 3D Reconstruction / Sensor Geometry

Distortion-Aware Gaussian Splatting under Satellite and Non-Pinhole Camera Geometry

A geometry-aware 3D reconstruction system for settings where standard pinhole camera assumptions break down. The project implements analytic camera Jacobians and distortion grids for satellite RPC, pushbroom, fisheye, and equirectangular cameras, then exposes a CPU-safe Hugging Face demo for inspecting how projection error affects Gaussian Splatting. The goal is to make novel-view synthesis more faithful to the actual sensor geometry rather than hiding distortion in preprocessing.

Generative AI / Remote Sensing / Climate Downscaling

Physics-Constrained Rectified Flow for Earth Observation Super-Resolution and Climate Downscaling

A conditional generative model for downscaling Sentinel-2 imagery, precipitation, and climate fields without losing physical consistency. The system combines a flow-matching velocity objective with residual penalties for conservation behavior, wind-field divergence, and spectral index consistency such as NDVI, NDWI, and SAVI. The demo includes AOI selection, variable controls, coarse-to-fine visualization, and a physics-violation dashboard so the output can be evaluated beyond visual sharpness.

GitHub 🤗 Hugging Face Paper PDF

Computer Vision / 3D Segmentation / Foundation Models

Geodesic Prompt Propagation for Open-Vocabulary 3D Scene Segmentation

A promptable 3D scene segmentation pipeline that starts from monocular video rather than assuming pre-built meshes or RGB-D scans. GeoSAM-3D reconstructs a Gaussian scene with monocular depth priors, lifts SAM-style masks into per-Gaussian feature embeddings, and propagates labels through a learned-feature plus heat-method geodesic kernel. The emphasis is on respecting scene topology, occlusions, and concavities when transferring 2D prompts into 3D.

GitHub 🤗 Hugging Face Paper PDF

Information Retrieval / NLP / Maritime Analytics

Language-Guided Trajectory Search over AIS Tracks and Satellite Imagery

A retrieval and reasoning interface for maritime analysts who want to search vessel behavior with natural-language prompts. The system aligns AIS trajectory features with text through a trajectory-CLIP encoder, connects the retrieval step to TGARD rendezvous reasoning, and pairs the selected tracks with Sentinel-2 chips and segmentation masks. It is meant to bridge open-vocabulary search, trajectory gap analysis, and visual inspection in one auditable workflow.

GitHub 🤗 Hugging Face Paper PDF

Multimodal ML / Remote Sensing / Anomaly Detection

Multi-Modal Dark Vessel Detection with AIS, SAR, Optical Imagery, and Trajectory Reasoning

An end-to-end maritime domain awareness stack for cases where AIS is missing, spoofed, jammed, or inconsistent with what satellite sensors observe. The project fuses Sentinel-1 SAR, Sentinel-2 optical imagery, AIS streams, and trajectory reasoning modules derived from TGARD and Pi-DPM. It frames dark vessel detection as a multi-modal decision problem that combines object detection, segmentation, change detection, anomaly reasoning, and an analyst-facing explanation trace.

GitHub 🤗 Hugging Face Paper PDF

Geospatial Systems / Microservices / MLOps

Map-Constrained Pickup and Drop-Off Pin Selection for Autonomous Ride-Hail Systems

A production-style microservice for choosing safe pickup and drop-off pins under road-network, curb, HD-map, and congestion constraints. The service generates H3 candidates around a request, filters them with map feasibility rules, scores them with gradient-boosted models, and exposes the decision path through gRPC, OpenTelemetry, and Prometheus metrics. The project demonstrates how spatial reasoning moves from a model output into a reliable operational service.

GitHub 🤗 Hugging Face Paper PDF

Reinforcement Learning / Preference Optimization / Trajectory AI

Physics-Informed Group Relative Policy Optimization for Trajectory Reasoning

A reinforcement-learning stack for training trajectory generation and reasoning policies while making hard physical violations costly. Pi-GRPO combines GRPO, PPO, and DPO trainers with a hybrid reward that includes kinematic-bicycle feasibility, jerk and curvature envelopes, and Pi-DPM reconstruction likelihood. The implementation emphasizes reward decomposition, rollout efficiency, preference data curation, and guardrails against reward hacking in spatial policy learning.

GitHub 🤗 Hugging Face

Multi-Agent Systems / Tool Use / Spatiotemporal Reasoning

Auditable Multi-Agent Spatiotemporal Reasoning for Trajectory Gaps and Rendezvous Detection

A multi-agent spatial reasoning system for decomposing natural-language questions into deterministic geospatial tools. GeoTrace-Agent routes work across AIS feeds, OSM road networks, Copernicus weather, Sentinel imagery, space-time prisms, STAGD gap detection, TGARD rendezvous search, and kinematic feasibility checks. The system is designed around typed plans, traceable tool calls, semantic caching, human review queues, and physically constrained outputs.

GitHub 🤗 Hugging Face

Geospatial Data Quality / LLM-Assisted Analytics / GIS

Distortion-Aware Geospatial Coordinate Correction with Deterministic and LLM-Assisted Analysis

An interactive correction tool for diagnosing and repairing distorted geospatial coordinates before downstream mapping or spatial analysis. The project combines a deterministic fallback engine with optional language-model analysis, clean projection rendering, and explicit reasoning about likely coordinate, datum, projection, and measurement-error issues. It is framed as a practical bridge between geospatial data cleaning and explainable spatial quality control.

Generative Modeling / Trajectory Forecasting / Urban AI

Geo-lucid Conditional Diffusion Models for High Physical Fidelity Trajectory Generation

Generated realistic synthetic vehicle trajectories for autonomous driving simulation and traffic prediction by co-designing GCDM, a two-stage cascaded diffusion model that injects attributed road-network priors through map-informed latent variables, spatial cross-attention, and a physics-based trajectory decomposition. The work targets trajectory generation that preserves both geometry and dynamics, producing stronger geo-distribution similarity and velocity, acceleration, and jerk fidelity than DiffTraj, Sashimi, DiffWave, and EETG on Porto and Harbin taxi data.

Graph Algorithms / Route Optimization / Electric Vehicles

Multi-Level Bi-Objective Routing for Pareto-Optimal Energy and Travel-Time Planning

Enabled real-time energy-vs-travel-time route planning for electric vehicles by co-designing MBOR with a boundary multigraph, Multi-level Encoded Pareto Frontier View, 2D cost-interval pruning, and multi-edge pruning. The method retrieves complete Pareto-optimal path sets over fragmented road networks and achieved more than 10x online-runtime improvement on real-world road network data, making multi-objective EV route planning feasible in interactive settings.

Trajectory Mining / Maritime Safety / Anomaly Detection

Physics-Based Trajectory Gap Detection and Possible Rendezvous Search

Reduced manual investigation areas for illegal maritime activity such as fishing, oil transfers, and trans-shipments by up to 80% through a multi-year research program on Space-Time Aware Gap Detection. The work combines comparison-less temporal indexing, R*-tree spatial indexing, maximal-union merge over space-time prism geo-ellipses, ellipse symmetry, bi-directional pruning, and early stopping. The STAGD, TGARD, and DC-TGARD line connects missing AIS reports to physically plausible search regions at the scale of roughly 500K ships.

Surrogate Modeling / Spatial ML / Sustainable Agriculture

Hybrid Spatial Neural Surrogate Models for Sustainable Agriculture Simulation

Accelerated expensive ecosystem simulations for sustainable agriculture with SM-Hybrid, a surrogate model for Daycent that models spatial autocorrelation and long-range tele-connections using hybrid spatial neural networks. It predicts land emissions, ecosystem fluxes, and crop yield across Iowa, Illinois, and Indiana about 2x faster than Daycent while improving accuracy over SM-ANN baselines, which supports faster scenario evaluation for climate-smart agriculture.

Biomedical AI / Spatial Deep Learning / Domain Adaptation

Spatially-Lucid AI Classification for Non-Euclidean MxIF Oncology Data

Supported precision immunotherapy by developing domain-adapted spatially-lucid neural networks for multi-category point set classification in non-Euclidean space. The work uses spatial ensemble learning, place-calibration parameters, weighted distance learning rates, spatial domain adaptation, spatial mix-up masking, and spatial contrastive predictive coding on MxIF oncology data to represent immune-tumor relationships in the tumor microenvironment.

GIScience / Mobility Analytics / Public Policy

Community-Engaged Mobility Analysis for COVID-19 Policy Response

Built an entity-relationship model, system architecture, and implementation for analyzing aggregated mobile device data during COVID-19 policy interventions. The project delivered county-level analytics on long-duration visits to high-risk business categories and fine-resolution device count maps to epidemiologists, analysts, and policymakers in collaboration with the Minnesota Department of Management and Budget, including outputs used in policy-facing mobility analysis.
